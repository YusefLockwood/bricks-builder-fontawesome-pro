== Description ==

Bricks Builder FontAwesome Pro plugin integrates Font Awesome icons into Bricks Builder. It includes support for Pro icons, custom weights, and families. The plugin provides an easy-to-use interface for selecting and customizing icons.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/bricks-builder-fontawesome-pro` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to 'Settings' -> 'Bricks FontAwesome' to configure the plugin.

== Frequently Asked Questions ==

= How do I get a Font Awesome API key? =

You can get a Font Awesome API key by signing up for a Font Awesome Pro account at https://fontawesome.com/.

= Can I use this plugin with the free version of Font Awesome? =

Yes, the plugin supports both the free and pro versions of Font Awesome.
